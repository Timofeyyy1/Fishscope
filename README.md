# Fishscope
В рамках развития рыботорговой системы fishery.ru возникла необходимость в создании внутреннего аналитического инструмента, который позволит маркетологам и аналитикам оперативно обрабатывать, агрегировать и анализировать ценовые данные по рыбной продукции из текстов объявлений, которые оформляются в свободной форме.
